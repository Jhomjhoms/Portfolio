# BSIT Software Portfolio

This is a professional portfolio template created for a BSIT student with a focus on software engineering, machine learning, and data-driven applications. It includes Home, About, Skills, Projects, and Contact sections.

How to use

- Open `index.html` in your browser.
- Edit content (name, contact details, project links) directly in `index.html`.
- Styles are in `css/styles.css` and JavaScript in `js/script.js`.

Files

- `index.html` — main page
- `css/styles.css` — styles and theme
- `js/script.js` — interactivity (dark mode, animations, form)
